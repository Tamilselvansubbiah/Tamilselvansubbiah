About this Project

The International Space Station will be a permanent earth orbiting laboratory. By observing and collecting information from The Space Station, scientists will learn more about our home planet and earth. By collecting information, conducting experiments and manufacturing materials on orbit. The station will operate at an altitude of 355 to 460 kilometers (208 to 285 statute miles). This real time application displays the movements of the satellite from its orbit, it helps to display the movements for each second. This project is used to track the real time satellite and its location will be shown in website. The real time data is collected from API –Application Programming Interface. This project is going to be implemented by Html for structuring the web page,CSS for Styling and the Javascript for developing logics and fetching the ISS data from the API.
